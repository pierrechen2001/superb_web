歡迎查看共享文件

這是一個範例 HTML 文件，展示如何通過密碼保護的方式分享文件給客戶。

重要提示： 您可以將任何 HTML 文件放入 shared_files/ 資料夾中，然後在配置文件中設置密碼即可。

使用說明

  1. 將 HTML 文件放入 shared_files/ 資料夾（不在 public 資料夾中）
  2. 在 shared_file_config.json 中配置文件名和密碼
  3. 將連結 /shared_file/文件名.html 分享給客戶
  4. 客戶輸入密碼後即可查看文件

文件內容

您可以在這裡放置任何 HTML 內容，包括：

  • 產品說明文件
  • 報價單
  • 設計稿展示
  • 任何需要密碼保護的網頁內容

此文件採用深藍色科技風格設計，符合精湛資訊工作室的品牌視覺。
