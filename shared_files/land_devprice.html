地政估價與稅務系統 — MVP 專案報價 PWA 版本

一、專案定位 Project Vision

目標是打造一套「專業地政士等級的 SaaS 工具」，提供案件管理、稅務試算、地籍圖整合、實價行情查詢、OCR 辨識謄本、自動估價（AVM）、專業 PDF 匯出等核心功能。

第一階段採 PWA 推出，讓使用者不需上架、不需等待，可直接在手機、平板、電腦瀏覽器啟用服務，加速市場驗證與流量成長。

等使用者數量達到一定基數後，再升級 iOS + Android 雙平台 Flutter App。

二、MVP 功能模組

軟體基礎

  • 會員登入 / 註冊（Email + Token 驗證）
  • 案件管理模組（建立、儲存、查詢）
  • 後端 API（Cloud Run）＋資料庫（MySQL / Postgres）
  • 系統架構設計（Architecture）＋ UI Wireframe
  • 上線部署流程（CI/CD, Domain, SSL, CORS）

稅金計算

  • 完整稅務試算邏輯（地價稅 / 房屋稅 / 契稅 / 土增稅/ 房地合一稅）
  • AVM 基礎模型（距離權重、時點平移、坪價調整）
  • 實價登錄行情查詢（串接政府開放資料）＋地圖連動

文件與圖資整合

  • 地籍圖（NLSC WMTS）＋空照圖 Overlay
  • OCR 謄本辨識（地號、面積、自動帶入欄位）
  • PDF 報告生成

四、額外費用 依需求選擇

系統上線後，需要持續運作的基礎設施費用，可依照實際需求選擇方案。大部分服務採訂閱制，從免費方案到每月約 2,000 元都有選擇。

註：以下費用為月付訂閱制，可隨時依需求升級或降級方案。

主要費用項目

項目 說明 費用範圍（月） 主要差異
網域租借 網站網址（如：yourdomain.com） 免費～300 元 免費方案通常為子網域，付費可擁有獨立網域
雲端資料庫 資料儲存與管理服務 免費～800 元 免費方案有容量與連線數限制，付費方案提供更高效能與備份
伺服器費用 系統運作所需的雲端主機 免費～1,200 元 免費方案有流量與資源限制，付費方案支援更高流量與穩定度

建議方案

若確定要長期發展，不建議選用免費方案。建議初期即選用大型可靠的雲端平台，以確保系統穩定、效能與可擴展性。

我們的推薦平台：

  • GCP（Google Cloud Platform）：Google 提供的完整雲端服務，穩定可靠
  • Northflank：現代化的部署平台，操作簡便、效能優異

這些平台提供完整的基礎設施服務（伺服器、資料庫、網域等），初期費用約每月 500–2,000 元，可依實際使用量彈性調整。
